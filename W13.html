Programming History

Lanuage Fact
HTML HTML (Hypertext Markup Language) was created by Tim Berners-Lee in 1991 while working at CERN.
JavaScript JavaScript was created by Brandom Eich while working at Netscape.
Python Python was created by Guido van Rossum and released in 1991.
C++ C++ was developed by Bjarne Stroustrup at Bell Labs as an extension od the C programming language.
Java Java was developed by James Gosling at Sun Microsystems and first released in 1995.
